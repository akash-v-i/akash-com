# Akash VI Portfolio

A premium personal developer portfolio website featuring a contribution-style activity grid animation revealing the name 'AKASH VI' as green pixel lettering.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
